# MILESTONES DEL PROYECTO

## MILESTONE 0: ASENTAR LAS BASES Y MODELIZAR EL PROBLEMA.
- Objetivos: Realizar un análisis completo de las historias de usuario tratando de entender el problema para un correcto diseño de los modúlos, expresar que estructuras de datos vamos a usar para tratar con los 
datos y elección del lenguaje usado para la codificación.
- Producto minimo viable: Un modelo estructurado codificado en el lenguaje seleccionado.
- Verificación: Será viable si lo aprueba el product manager.
- HU relacionadas: HU1, HU2, HU3.

## MILESTONE 1: IMPLEMENTAR ALGORITMO DE CREACIÓN DE HORARIOS BÁSICOS.  
- Objetivos: Crear un algoritmo capaz de cumplir unos requisitos básicos que son que los días libres de cada semana deben ser correlativos y no debe de haber más de tres semanas en el mismo turno.
- Producto minimo viable: Un modelo codificado con la capacidad de crear estos horarios.
- Verificación: Será viable si se cumplen los dos requisitos básicos.
- HU relacionadas: HU1, HU2, HU3.
